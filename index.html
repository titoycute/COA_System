🛠️ Google Drive Setup

Files upload to your Drive folder via a free Google Apps Script. One-time setup, takes ~3 minutes.

1

Go to script.google.com and click New project

2

Delete all default code, then paste the Code.gs file content (download below). Click 💾 Save.

3

Click Deploy → New deployment. Choose type: Web app. Set Execute as: Me and Who has access: Anyone. Click Deploy and authorize when prompted.

4

Copy the Web app URL shown (looks like https://script.google.com/macros/s/ABC…/exec)

5

Paste the URL below and tap Save & Test

⚠️ The folder 1fIVbZvFBE9AXrcyTHZVYC0vh3iBfJmLJ is already hardcoded in the script. Make sure the Google account you use owns or has Editor access to that folder.
